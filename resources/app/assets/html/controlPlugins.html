La recherche d'une règle se fait séquentiellement dans tous les plugins de votre bibliothèque, les uns après les autres. Par conséquent, le temps de recherche de la règle et le traitement de l'action associée peut devenir très long suivant le nombre de plugins et de règles que vous avez. Pour accélérer les temps de réponses pour les plugins les plus utilisés, vous pouvez leurs donner un ordre de chargement pour que leurs règles soient trouvées en premier. Choisissez un plugin puis commencez par la position 1, puis un 2ème plugin et 2, puis un autre et 3, etc... Laissez la valeur "Par défaut" pour qu'un plugin soit chargé aléatoirement sans ordre défini. Pour supprimer une position, enlevez la valeur. Vous pouvez aussi définir si le plugin est actif ou non.

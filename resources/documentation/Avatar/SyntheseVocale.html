Synthèse Vocale

La synthèse vocale est la voix que vous utilisez pour les dialogues d'Avatar Client.

Par défaut, Avatar utilise la voix installée pendant l'installation du client à l'étape 3.3: Microsoft Speech Plateform - Runtime language (version 11)].

Avatar vous propose une voix supplémentaire Virginie (très féminine) disponible dans le répertoire voice du client.
Vous pouvez installer la voix Virginie.


Vous pouvez aussi demander sur la communauté un lien de téléchargement pour une voix masculine comme Sébastien

Important: 
Pour une plateforme 64 bits, configurez la voix installlée en utilisant le sapi 32 bits:

  1. Cliquez sur le menu Démarrer puis dans la barre de commandes Windows la commande suivante:
                            %windir%\SysWOW64\speech\SpeechUX\sapi.cpl
  2. Dans la boite de dialogue, choisissez Virginie.

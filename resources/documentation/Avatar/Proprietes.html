Propriétés

Les paramètres suivants définissent la configuration de la confidence du mot-clé déclencheur de l'écoute.


Paramètre

Type

Valeur par défaut

Description

hotWord -> default_threashold

String

"0.900"

Confidence basse par défaut pour le mot-clé.
Définissez une valeur à 3 chiffres après la virgule.
Note: Non utilisé dans cette version.

hotWord -> max_threashold

String

"0.918"

Le client est prévu pour que la confidence du mot-clé s'incrémente automatiquement lorsque le nombre de faux positifs définis dans la propriété increased_threashold est atteinte (voir ci-dessous). 

Cette propriété définie la confidence maximale à ne pas dépasser pour l'incrément automatique.

Définissez une valeur à 3 chiffres après la virgule.
 
Note: Si vous ne connaissez pas la confidence maximale à positionner ici, reporte-vous au chapitre Réglage.

Pour revenir à une confidence basse:

  • Si la confidence est devenue trop haute et que le client ne vous entend plus, vous devez le faire manuellement en abaissant la valeur de confidence dans le fichier c:\Dossier d'install du client\hotword\hotword.xml
      • Ouvrez le fichier
      • Repérez la ligne <tag>out.action._attributes.threashold="0.928"
      • Modifiez la valeur, par exemple: <tag>out.action._attributes.threashold="0.915"

hotWord ->  increased_threashold

Integer

3

Le nombre de faux positifs acceptables avant que la confidence ne s'incrémente automatiquement. Lorsque la valeur maximale définie dans la propriété max_threashold est atteinte, l'incrément automatique est stoppé.

Pour que l'incrément soit automatique:

Lorsque le mot-clé déclenche l'écoute, ne dite rien. Après le délais défini dans le paramètre listen -> timeout, l'écoute est stoppée et le nombre de faux positifs augmente. Lorsque la valeur atteint le nombre défini ici, la confidence s'incrémente automatiquement et Avatar vous le signifie vocalement.

hotWord ->  increased_threasholdBy

String

"0.001"

Le pas d'incrément de la confidence du mot-clé.
Définissez une valeur à 3 chiffres après la virgule.


Plus la valeur est basse, plus le réglage de la confidence est précis. La valeur par défaut est correctement ajustée pour un réglage précis.

Accueil


Avatar est un assistant de reconnaissance vocal nodejs open source client-serveur dédié à la conception d'une maison intelligente et à l'internet des objets.


Avatar reprend historiquement les concepts de l'assistant SARAH - (Home Automation). Néanmoins, Avatar et SARAH ne sont pas compatibles. Vous pouvez aussi regarder la documentation de développement pour savoir comment migrer un plugin SARAH vers Avatar.


Avatar est un assistant vocale nativement multi-rooms: 

  • Il ne nécessite aucun ajout d'applications supplémentaires, vous disposez des clients dans les pièces à contrôler, c'est tout. Le serveur sait exactement quelle pièce a envoyée l'action, l'exécute pour le client et lui retourne le résultat dans tous les cas de figures, y compris dans les dialogues de questions-réponses (les askme) et jusqu'à la répétition de la dernière action réalisée (qui, bien sûr, doit être associée au client). 


Avatar traite et analyse les règles par Traitement Naturel du Langage (NLP):

  • Le principe est de dicter une règle via un client qui est ensuite envoyée au serveur. Cleui-ci la traduit en Anglais et l'analyse par NLP. Si une intention est trouvée, le serveur déclenche ensuite une action associée par requète HTTP. Ces scripts interagissent avec tous les objets connectés (box domotiques, TV, montres, lampes, gadgets, …) ou l’open data (horaires de cinéma, programmes TV, météo, horaires de bus, wikipedia…).


Avatar intègre de nombreux automatismes, comme par exemples:
    - La possiblité d'avoir des clients virtuels nomades connectés au serveur (une montre, un téléphone, une tablette, etc...).
    - Une gestion complète du volume des périphériques lors d'un dialogue.
    - La possibilité d'enchainer plusieurs règles sans dire systématiquement le mot-clé d'accroche.
    - L'intégration de votre système de son Wifi Sonos System comme speaker.
    - Une gestion automatique de la confidence.
    - Un mode intercom entre les clients.
    - Un redémarrage des Avatar clients à la demande et beaucoup d'autres choses...


Avatar fonctionne sur Microsoft Windows:

    1. Avatar Serveur est une application nodejs mais incluant quelques modules C qui ont dû être compilés. Comme la compilation a été faite sur une plateforme Microsoft Windows, il ne peut donc être exécuté que sur cette plateforme. Une utilisation sur un autre type d'OS (e.g. Unix/Linux) est  possible en recompilant les modules C de l'application sur la plateforme cible.
    2. Avatar Client est une application nodejs mais utilisant la technologie de reconnaissance vocale de Microsoft Speech Engines. Il ne peut donc être utilisé que sur ce système d'exploitation. Mais en trouvant une autre approche pour gérer le mot-clé déclencheur de l'écoute, rien n'empêche de migrer Avatar Client vers un OS Unix/Linux.

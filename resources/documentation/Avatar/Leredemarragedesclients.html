Le redémarrage des clients


Vous pouvez utiliser cette action pour redémarrer les clients à la demande.

Important:
Cette commande nécessite le paramètrage d'un fichier de propriétés afin de connaitre le type de microphone utilisé pour le client Avatar.

Ce paramètrage est a réaliser pour chaque client:

  • Ouvrez le fichier c:\Dossier d'install du client\app\restart\config.json
  • Si besoin, modifiez la propriété nodejs, laissez les autres propriétés avec les valeurs par défaut.


Propriété

Valeur par défaut

Description

nodejs

Avatar_Client_Microphone.cmd

Défini le fichier de lancement utilisé pour démarrer le client.

Entrez une des deux valeurs possibles, soit:

  • Microphone classique: Avatar_Client_Microphone.cmd
  • Kinect: Avatar_Client_Kinect_Audio.cmd

stopGracefully

true

Défini si la tâche système du client doit être fermée proprement ou "tuée".

timeBeforeKill

20000

Lorsque la tâche système du client Avatar ne peut être fermée proprement, elle est "tuée" après le laps de temps défini dans cette propriété (en ms)

minimizeServerminimizeServer

false

Non utilisé dans cette version

hideServer

false

Non utilisé dans cette version



Syntaxe de phrase à prononcer: [Déclinaison de Redémarrer] * [Tous || Client Name]

Exemples:

  • Vous: Redémarre
    • Redémarre la pièce courante.
  • Vous: Redémarre le Salon
    • Redémarre le client Salon
  • Vous: Tu peux redémarrer la Chambre
    • Redémarre le client Chambre
  • Vous: Pourrais-tu redémarrer tous les clients
  • Vous: Redémarre tous les clients
    • Redémarre tous les clients connectés

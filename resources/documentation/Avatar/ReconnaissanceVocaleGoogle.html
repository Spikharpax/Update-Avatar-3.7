Reconnaissance Vocale Google


La dernière version du client Avatar Windows utilise la reconnaissance vocale par le Web Speech Api de Google Chrome.

Afin de l'utiliser, la configuration d'un serveur Web HTTPS est nécessaire.


Pré-requis:

Google Chrome doit être installé sur le PC client. Téléchargez-le ici.


L'application est configurée pour utiliser le nom de DNS avatar.ia.fr

  1. Afin que ce nom soit reconnu, il est nécessaire de l'ajouter dans le fichier hosts de Windows:
    1. Ouvrez un gestionnaire de fichier
    2. Déplacez-vous dans le répertoire <Windows>\System32\drivers\etc
    3. Ouvrez le fichier hosts dans un éditeur de texte
    4. Ajoutez le nom de domaine avatar.ia.fr comme dans l'exemple ci-dessous:



Note: Si un caractère # existe en début de ligne (avant le 127.0.0.1), enlevez-le pour que la ligne ressemble exactement à l'exemple ci-dessus.


    1. Sauvegardez le fichier
      1. Il est possible que vous n'ayez pas les droits administrateur pour modifier ce fichier. Si vous utilisez notepad++, il peut vous demander si vous voulez passer en mode administrateur afin de le sauvegarder, répondez oui.
      2. Dans tous les cas, il existe toujours un moyen pour sauvegarder le fichier hosts avec la nouvelle entrée. Comme par exemple ouvrir une fenêtre de commande DOS en mode administrateur, se déplacer dans le répertoire du fichier et de faire un

echo 127.0.0.1   localhost  avatar.ia.fr >> hosts

      1. Le cas échéant, faites une recherche sur Google pour consulter des posts ou demandez de l'aide sur la communauté.



  1. Ajoutez un certificat de confiance dans les paramètres de Chrome
    1. Démarrez Chrome
    2. Allez dans les paramètres
    3. En bas de la page des paramètres, cliquez sur Paramètres avancés
    4. Cliquez sur Gérer les certificats
    5. Dans la fenêtre Certificats, Cliquez sur l'onglet Autorités de certification racines de confiance
    6. Cliquez sur Importer...
    7. Cliquez sur Suivant puis dans la fenêtre suivante, cliquez sur Parcourir...
    8. Déplacez vous dans le répertoire <Avatar Windows Client>\app\node_modules\lib\certificats
    9. Sélectionnez avatar.ia
    10. Cliquez sur Suivant, Suivant, Terminer
    11. Répondez Oui au message d'avertissement
    12. Fermez la boite de dialogue


  1. Important: Lors de la première connexion à Avatar, le navigateur vous demandera d'accepter que l'application utilise le microphone.Vous répondrez alors oui.

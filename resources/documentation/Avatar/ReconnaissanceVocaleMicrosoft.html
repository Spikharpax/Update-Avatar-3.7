Reconnaissance Vocale Microsoft

La reconnaissance vocale Microsoft Speech Engines est utilisée pour le mot-clé que vous avez choisi pour déclencher une écoute du client et uniquement pour celui-ci. Ensuite, la reconnaissance vocale par le Web Speech Api de Google Chrome prend la relève pour enregistrer votre règle.


Pourquoi utiliser la reconnaissance vocale Microsoft Speech Engines ?

Il faut comprendre qu'il est impossible que le client Windows Avatar écoute constamment ce que vous pouvez dire sinon il passerait son temps à comprendre des ordres fantômes, ce qu'on appelle des "faux positifs".


Par conséquent, il faut un mot-clé d'accroche qui vous permette d'activer l'écoute et dicter votre règle.

Pour un client Windows Avatar, ce mot-clé d'accroche est défini en utilisant la reconnaissance vocale Microsoft Windows locale du PC.


Sur chaque PC où est installé un Avatar Client, vous pouvez donc entrainer la reconnaissance vocale de Microsoft Windows pour une meilleure compréhension de votre voix.


Si vous êtes plusieurs à utiliser Avatar dans la maison, dans un 1er temps, vous pouvez faire un test de chacun des intervenants pour vérifier si la reconnaissance du mot-clé est correcte pour chacun sans entrainement.
Par la suite, vous pourrez toujours revenir à ce chapitre et créer un profil de reconnaissance en effectuant un entrainement. Malheureusement, comme l'entrainement est associé à un profil et que ce profil est courant, il sera amélioré pour celui qui a fait l'entrainement mais pas pour les autres intervenants.


Astuce:

Si vous êtes plusieurs, comme la reconnaissance Windows est nativement meilleure avec les voix masculines, préférez entrainer une voix féminine, vous aurez ainsi une meilleure compréhension pour les 2 voix.


Pour entrainer la reconnaissance vocale Windows:

  1. Entrez reconnaissance vocale dans la zone de recherche du menu Démarrer, puis cliquez sur Reconnaissance vocale.

ou


  1. Ouvrez la Reconnaissance vocale en cliquant sur le bouton Démarrer
  2. Cliquez sur Panneau de configuration, sur Options d’ergonomie, puis sur Reconnaissance vocale


Ensuite


  1. Cliquez sur Exécuter le module d’apprentissage de votre voix
  2. Suivez les instructions qui s’affichent à l’écran

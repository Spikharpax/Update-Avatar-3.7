Redémarrage automatique

Le paramètre suivant définie le redémarrage automatique


Paramètre

Type

Valeur par défaut

Description

lost_connection -> restart

Integer

5

Propriété exprimée en minutes définissant le délai pour que le client redémarre automatiquement si la connexion avec le serveur est perdu et que le client ne peut pas se reconnecter automatiquement.


Il peut arriver que la perte de connexion soit liée aux sockets réseaux, dans ce cas, la reconnexion automatique est impossible.  Le client redémarre alors automatiquement après ce délais.


Si le client est dans l'incapacité de redémarrer automatiquement, Avatar Serveur peut vous envoyer une notification sur votre téléphone. Reportez-vous au chapitre Notification pour plus de détails.



Nouveauté de la version 3.7 !

Le paramêtre au-dessus n'est plus nécessaire, AVATAR Client peut maintenant redémarrer automatiquement même si la connexion est intérrompue.


Pour cela, ajoutez le petit programme AutomaticRestartClient : Téléchargez ici !


Dézipez le projet puis copiez le fichier controlStarted.vbs dans le répertoire principal de votre client AVATAR, par exemple: C:\AvatarClient\
(Là où se trouve le fichier AvatarWebAPIClient.exe)


Ensuite:
1: Ouvrez les tâches planifiées : Menu Windows _> Rechercher -> Planificateur de tâches
2: Dans le menu à droite, cliquez sur "Importer une tâche"
3: Sélectionnez la tâche (fichier ControlAvatarClientStarted.xml)
4: Dans l'onglet "Action", cliquez sur l'action de la tâche
5: Modifiez le chemin du fichier "controlStarted.vbs" que vous avez téléchargé en le sélectionnant dans votre répertoire AVATAR Client par le bouton "Parcourir..."
6: Modifiez le chemin "Commencer dans..." par votre répertoire AVATAR
7: Modifiez le compte utilisateur pouvant exécuter la tâche en cliquant sur le bouton "Utilisateur ou Groupe" dans l'onglet Général
8: Dans la fenêtre de sélection, entrez le nom du compte utilisateur qui démarre le PC (Surement le votre...) puis cliquez sur "Vérifier les noms". Le nom choisi apparait ensuite dans la fenêtre.
9: Cliquez sur Ok pour sauvegarder le nom.
10: Cliquez ensuite sur le bouton Ok dans la fenêtre principale pour enregistrer la tâche.
11: Retrouvez la tâche "ControlAvatarClientStarted' puis clique droit et "Activer"


Vous avez maintenant un client qui ne s’arrêtera jamais.
Un contrôle de l'exécution du client est fait toutes les 2 minutes.
Si vous avez un démarrage automatique du client AVATAR au démarrage du PC (dans shell:startup), supprimez obligatoirement le raccourci sinon 2 clients seront démarrés au boot du client.

Pour info, si vous voulez arrêter temporairement le redémarrage du client sur votre pc, ouvrez le planificateur de tâches, cliquez sur la tâche, clic droit et "Désactiver". Lorsque vous voudrez le réactiver, cliquez sur "Activer"

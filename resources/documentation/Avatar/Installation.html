Installation

Pour installer un client Windows, suivez la procédure suivante:


Étape 1: Installation du Core de l'application

    1. Sur le PC où sera installé un client:
    2. Téléchargez Avatar Windows Client depuis le projet GitHub dans un répertoire temporaire, par exemple:
      • C:\tmp\Avatar_client
    1. Dézippez le fichier Avatar-Client-master.zip dans ce répertoire temporaire
    2. Créez un répertoire où sera installé le client, par exemple:
      • C:\Avatar\Client
    1. Déplacez-vous dans le répertoire C:\tmp\Avatar_client\Avatar-Client-master
    2. Copiez le fichier Avatar-Windows-Client.zip extrait du zip dans son répertoire d'installation créé à l'étape 4
    3. Dézippez Avatar-Windows-Client.zip dans ce répertoire
      1. Prenez soins de supprimer le dernier répertoire qui est proposé dans l'extraction pour ne garder que C:\Avatar\Client




Étape 2: Vérifier les mises à jour

Vérifiez si il existe une mise à jour à installer après le core de l'application.


    1. Vérifiez la version courante sur la page principale du projet GitHub
      • Par exemple, supposons que la version courante sur la page principale est 3.3
    1. Vérifiez ensuite dans le répertoire temporaire où le fichier Avatar-Client-master.zip a été dézippé, si il existe un répertoire avec le numéro de la version courante:
      • Par exemple,  C:\tmp\Avatar_client\Avatar-Client-master\3.1
    1. Si un répertoire de mise à jour existe, suivez alors la procédure suivante pour l'installer sinon passez à l'étape 3:
      1. Si il existe un fichier README.txt dans ce répertoire de mise à jour, lisez-le avant de continuer. Il est possible que certaines choses doivent être réalisées avant de poursuivre, si tel est le cas, effectuez la procédure du README.
      2. Ensuite, déplacez-vous dans le répertoire de la mise à jour
      3. Copiez le contenu
      4. Collez le contenu dans le répertoire où est installé le client.
        • Dans l'exemple de l'étape 1.4: C:\Avatar\Client
        • Si nécessaire, acceptez les remplacements de fichiers



Étape 3: Installation des applications connexes

    1. Téléchargez et installez Microsoft .NET Framework 4.5
          • Dans un environnement récent comme Windows 10, ce framework est déjà installé. Dans tous les cas, le programme d'installation vous le signifiera.
    1. Téléchargez et installez la version x86 de Microsoft Speech Plateform - Runtime (Version 11)
    2. Téléchargez et installez Microsoft Speech Plateform - Runtime language (version 11)
      • Pour le Français, vous pouvez sélectionnez la voix MSSpeech_TTS_fr-FR_Hortense.msi



Si vous disposez d'une Kinect pour Windows V1 ou V2, vous devez installer les applications associées sinon vous pouvez passer à l'étape de Reconnaissance Vocale.


Étape 4: Uniquement si vous avez une Kinect V1 ou V2

Pour une Kinect V1:

    1. Téléchargez et installez Kinect for Windows SDK v1.8
    2. Téléchargez et installez Kinect for Windows Language Packs v11.0
      • Choisissez le language FR-fr lors du téléchargement

Pour une Kinect V2:

    1. Téléchargez et installez Kinect for Windows SDK V2.0
    2. Téléchargez et installez Kinect for Windows Language Packs v11.0
      • Choisissez le langage FR-fr lors du téléchargement


Passer à l'étape de Reconnaissance Vocale.

Phrases du client

Les paramètres suivants définissent les phrases prononcées par le client suivant le contexte de dialogue.


Toutes ces phrases sont bien sûr modifiables, vous pouvez aussi en ajouter.


Paramètre

Valeur par défaut

Description

locale -> fr-FR -> tts_restoreContext

"Oui ?|Je t'écoute|Hume ?|yesse ?"

Après avoir restauré le mode écoute avec le mot-clé déclencheur, le client vous signifie qu'il vous écoute en disant un des textes aléatoirement entre les pipes ('|')

locale -> fr-FR -> tts_restart_restoreContext

"Autre chose ?|Je peux faire autre chose ?|Tu veux autre chose ?"

Associé au mode loop.


Après la première règle que vous avez passée, le client vous demande si vous désirez autre chose en vous disant un des textes aléatoirement entre les pipes ('|')


Note: Ne fonctionne que si le mode loop est activé (voir listen -> loop_mode)


locale -> fr-FR -> answers_forceMute

"Pardon|j'ai cru, désolé|Sorri|Excuse"

Si vous avez stoppé l'écoute en disant un des textes de la propriété "tts_forceMute", le client vous retourne un message de validation en disant un des textes aléatoirement entre les pipes ('|')


locale -> fr-FR -> answers_cancel

"d'accord|reçu|pas d'soucis"

Si vous avez stoppé l'écoute en disant un des textes de la propriété "tts_cancel" le client vous retourne un message de validation en disant un des textes aléatoirement entre les pipes ('|')

locale -> fr-FR -> answers_thank

"avec plaisir|à ton service"

Si vous avez stoppé l'écoute en disant un des textes de la propriété "tts_thank" le client vous retourne un message de validation en disant un des textes aléatoirement entre les pipes ('|')

locale -> fr-FR -> restart

"recommence..."

Retour prononcé par Avatar Client lorsqu'une phrase n'a pas été correctement comprise avant l'envoi au serveur Avatar

locale -> fr-FR -> increase_threashold

"Je met ma confidence à..."

Après le nombre de faux-positifs définis dans la propriété hotWord ->  increased_threashold (voir les propriétés de confidence et mot-clé), la confidence est automatiquement incrémentée et Avatar prononce cette phrase pour vous le signifier.

locale -> fr-FR-> max_restart

"Excuse-moi|Pardon"

Après le nombre maximum de redémarrage de l'enregistrement après une incompréhension d'Avatar défini dans la propriété listen ->  max_restart (voir les propriétés de l'écoute),l'écoute est stoppée et Avatar prononce cette phrase pour vous le signifier.

locale -> fr-FR -> max_mute

"Je suis trop bavarde je crois. Je me met en silence."

Après le nombre de maximum possibles enchainés (qui ont arrétés l'écoute) défini dans la propriété listen ->  max_mute (voir les propriétés de l'écoute), le client se met en mode silence complet et Avatar prononce cette phrase pour vous le signifier.
